## dependencies

 @babel/preset-env
 @babel/preset-react
 babel-loader
 react
 react-dom
 react-hot-loader"
 webpack
 webpack-cli
 webpack-dev-server
 bootstrap
 css-loader
 style-loader
`

## install

### write in console
`git clone  https://github.com/Avazkhon/library`

### enter folder
`cd library`

### install download dependencies
`npm i`

### start SPA
`npm start`
